# SSC0903_CAD_Trab1
Projeto do primeiro trabalho da disciplina de Computação de Alto Desempenho da USP São Carlos (1º semestre de 2026).

## Execução
Para compilar todos os códigos    : make all

Para executar o código sequêncial : make runseq input=entradas/input.txt (escolher arquivo de entrada)

Para executar o código paralelo   : make runpar input=entradas/input.txt (escolher arquivo de entrada)

Se rodar o script run_all.sh, ele vai executar automaticamente ambos programas com todos os arquivos de entrada. Esses tempos sao registrados nos arquivos da pasta "saida".
